### Run Proyect

1- Crear la BD **traceability** 

2- Ejecutar el script **scriptcreaciontablas.sql** en la BD

3- Ingresa al Directorio del proyecto

    $ cd traceability 

4- Ejecuta script de inicio

    $ ./run.sh
